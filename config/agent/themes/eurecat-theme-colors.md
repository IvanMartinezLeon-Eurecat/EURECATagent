## Notas

- `secondary` y `dim` están definidos como colores ANSI y aquí se muestran con sus equivalentes aproximados en escala de grises xterm.
- `text` está configurado como cadena vacía, así que este documento lo mantiene como token lógico en lugar de forzar un valor hexadecimal.
- Estas vistas previas usan HTML dentro de Markdown para que el fichero se parezca visualmente al tema del terminal al renderizarse en un navegador o visor Markdown con soporte HTML.
- Las demos ANSI están pensadas para ejecutarse en un intérprete de comandos, no para renderizarse directamente en visores Markdown.
- Se mantienen en inglés los nombres de tokens, comandos y etiquetas literales de Pi cuando representan texto real de la interfaz.
